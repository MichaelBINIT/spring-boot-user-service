# Spring Boot User Service

Ein einfaches Spring-Boot-Projekt zur Verwaltung von Benutzern.

## Funktionen

- GET User
- GET All Users
- POST Add User
- DELETE User

## Technologien

- Java
- Spring Boot
- Spring Data JPA
- Gradle
- H2/MySQL
- wsl/Ubuntu/docker

## REST-Endpunkte

GET    /users/getUser/{userName}
GET    /users/all
POST   /users/addUser
DELETE /users/delUser/{userName}
